- 需求分析
    - 编写用户故事
    - 撰写需求分析
- 商业分析
    - 撰写创业计划书
- 功能清单
    - 编写功能清单

## 研发工程师

- 代码审核
    - 代码审核和优化建议
- 设计模式
    - 底层架构设计
    - 功能代码实现
- 技术文档
    - 安全报告
    - 性能优化
    - 学习文档
