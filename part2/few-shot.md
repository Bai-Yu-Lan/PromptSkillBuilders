# 少样本提示

少样本提示词主要是以向大模型举例子提问的方式

- 输入

```
“7月29日，第三方市场研究机构Canalys公布了2022年第二季度中国智能手机出货量情况。受到疫情影响，大家的购机热情没有以往高涨，今年第二季度智能手机出货量同比下降了10%，出货量仅为6740万部。其中，vivo出货量为1320万台，排名第一；荣耀紧随其后，出货量为1300万部；OPPO（包括一加）以1180万部的出货量排在第三名；小米以1060万台的出货量排名第四；苹果排名第五，出货量为990万台。在手机市场整体疲软的背景下，今年第二季度，手机厂商们的出货量呈现出了两位数的下滑，只有荣耀和苹果实现了逆势增长。其中，荣耀年增长率达到88%，可以说过于夸张。苹果年增长率也达到了25%，较为稳健。”
vivo【中性】：虽然出货量第一，但是出货量有一定的下滑
荣耀【正面】：销售量仅此于vivo，且实现了逆势增长
小米【中性】：出货量中规中矩
苹果【正面】：虽然出货量较少，但也实现了逆势增长

“在新能源汽车这样的增长势头下，2023年就不只是新能源汽车市场洗牌了，而是已经开始挤压燃油车型的生存空间。日系、德系的三大销冠车型轩逸、新朗逸、卡罗拉销量在年末甚至出现了腰斩现象，这意味着明年对于燃油车型更加艰难。德系、日系销量的腰斩，也将进一步影响合资厂，以上汽、一汽、广汽为代表的央企汽车集团主要利润都来自于合资车型的分润，德系日系倒下，他们的亏损也将是非常惊人的。新能源汽车的洗牌，已经洗到了深水区。比亚迪今年销量实现了2倍以上的增长，一家厂商的销量占比超过33%，这在国内汽车行业是创造历史的纪录，随着比亚迪走出国门，在多个地区的销量也取得了较好的成绩，这是产品竞争力的体现。在国内新能源汽车销量达到100%的增长情况下，吉利汽车、广汽埃安、奇瑞汽车、长安汽车、哪吒汽车、零跑汽车跑赢了大盘。而特斯拉、理想汽车、小鹏汽车、蔚来汽车、一汽大众、上汽大众都不尽如人意。尤其是南北大众曾是燃油车时代的领跑者，如今随着德系电动化转型较慢，自主品牌青黄不接，市场份额已经是零头，无疑面临着极大的转型压力。上汽通用五菱虽然凭借着min EV系列赚足了眼球，但没有新的爆款产品，增长以及停滞；而长城汽车新能源车型的销量增长已经陷入停滞，侧面说明其新能源化转型的严重滞后，与同段位的吉利、奇瑞形成了鲜明对比，已经出现了掉队的征兆。”
```

- 输出

```
正面情感：

比亚迪： 比亚迪在2022年实现了销量增长超过两倍，一家厂商的销量占比超过33%。这被描述为创造历史的纪录，并且被视为产品竞争力的体现。这表明比亚迪在新能源汽车市场上表现出色。

吉利汽车、广汽埃安、奇瑞汽车、长安汽车、哪吒汽车、零跑汽车： 这些厂商被称为在国内新能源汽车销量增长100%的背景下取得成功，成功地赢得了市场份额。这被视为他们在新能源汽车领域的积极表现。

若干新能源汽车公司： 在整个新能源汽车市场增长的大背景下，一些公司被认为是大盘的赢家，这在情感上被视为正面因素。

负面情感：

德系、日系车型（轩逸、新朗逸、卡罗拉）： 这些车型在提到它们的销量时出现了腰斩现象，意味着它们在新能源汽车市场中面临艰难。这被视为负面因素。

南北大众： 曾经是燃油车时代的领跑者的南北大众，如今随着电动化转型较慢，自主品牌竞争不佳，市场份额已经很小，面临着极大的转型压力。这被描述为负面情况。

特斯拉、理想汽车、小鹏汽车、蔚来汽车、一汽大众、上汽大众等： 这些厂商在提到它们时被描述为不尽如人意，可能在新能源汽车市场中面临挑战。

上汽通用五菱和长城汽车： 上汽通用五菱虽然引起了注意，但没有新的爆款产品，增长停滞。长城汽车新能源车型的销量增长也陷入停滞，显示出其新能源化转型滞后，与竞争对手形成鲜明对比，已经出现了掉队的迹象。这些情况被描述为负面因素。
```
